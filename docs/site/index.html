Features

60+ Apps Across All Platforms

A unified collection of applications spanning desktop, mobile, embedded, web, and cloud environments.

Desktop Apps

Full-featured desktop applications including eOffice, EoStudio, EoSim, and eBrowser for productivity and development.

5 Mobile Apps

Cross-platform mobile apps: eRide, eSocial, eTrack, eTravel, and eWallet — built with native performance.

40+ Native LVGL Apps

Lightweight GUI apps built with LVGL for embedded displays, touchscreens, and resource-constrained devices.

11 Browser Extensions

Chrome and Firefox extensions for developer tools, productivity, and EmbeddedOS ecosystem integration.

Cloud Services

Backend services for OTA updates, telemetry, user management, and cross-device synchronization.

Automated CI/CD Builds

GitHub Actions pipelines for automated building, testing, and publishing across all target platforms.

GitHub Pages App Store

Browse, search, and download apps from the self-hosted App Store powered by GitHub Pages.
